# BSPE Connect

Proprietary WordPress plugin for BSPE Legal Marketing client sites.

## Install

Download the latest `bspe-connect.zip` from the [Releases page](https://github.com/BSPE-Legal-Marketing/bspe-connect/releases/latest) and install it via **Plugins → Add New → Upload Plugin** in WordPress.

> **Always install from the Releases page.** Never use the "Code → Download ZIP" button — it produces a folder name WordPress doesn't expect.

After the initial install, updates appear automatically in WP admin → Plugins. Click **Update** to install.
